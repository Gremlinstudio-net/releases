# GremlinStudio Releases

Official release binaries for [GremlinStudio](https://gremlinstudio.net) — a powerful IDE for Azure Cosmos DB graph databases.

## Downloads

Head to the [**Releases**](https://github.com/Gremlinstudio-net/releases/releases) page to download the latest version for your platform:

| Platform | File |
|----------|------|
| Windows x64 | `GremlinStudio-win-x64.zip` |
| macOS (Apple Silicon) | `GremlinStudio-osx-arm64.zip` |
| macOS (Intel) | `GremlinStudio-osx-x64.zip` |
| Linux x64 | `GremlinStudio-linux-x64.zip` |

Each zip includes a `README.md` with platform-specific installation instructions.

## Quick Start

1. Download the zip for your platform from [Releases](https://github.com/Gremlinstudio-net/releases/releases)
2. Extract to a folder of your choice
3. Run `GremlinStudio.exe` (Windows) or `./GremlinStudio` (macOS/Linux)
4. Your browser opens automatically — connect to your Cosmos DB instance and start querying
